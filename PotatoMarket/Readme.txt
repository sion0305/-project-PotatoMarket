<< 실행 및 테스트 방법 >>

테스트 id : dongduk / pw : ddd

- 메인 화면(index) 
메인 화면에서는 각 카테고리 별 등록 최신순으로 4개씩 뽑아 총 12개의 신상품을 추천해준다. 
전체 목록을 볼 수 있으며, 각 카테고리별로 볼 수도 있다. 
모든 상품의 상품명 클릭 시, 해당 상품의 상세 화면으로 넘어가며, 직거래와 공동구매 상품은 이미지 클릭 시, 장바구니에 바로 담을 수 있다.
만약, 해당 상품이 판매 완료된 상품일 경우, alert 창을 통해 판매가 완료된 상품임을 알린다.

- 회원 가입 및 로그인 
사용자는 아이디, 비밀번호, 이름, 이메일, 전화번호 정보를 입력한 후, 모든 조건이 만족해야만 회원 가입이 가능하다. 
회원 가입과 취소 버튼을 누를 경우, 모두 로그인 창으로 이동하고 로그인 성공 시, 메인 화면(index)으로 넘어간다. 

- 회원 정보 수정 및 탈퇴
마이페이지에 있는 회원 정보 수정/탈퇴를 통해 회원 정보를 확인할 수 있다. 
수정은 아이디를 제외한 나머지 정보들을 수정할 수 있으며, 탈퇴 버튼을 통해 탈퇴가 가능하다.
(※ 주의 : 아이디를 수정할 경우, 모든 정보가 수정되지 않는다.)

- 상품 등록(직거래, 공구, 경매)
상품 등록은 로그인 했을경우에만 가능하다. 
상품 등록의 모든 입력 항목과 이미지는 필수 사항이며, 가격은 100원 이상, 최대 인원은 2명 이상부터 가능하고, 
공동구매의 마감일과 경매의 종료일은 현재 날짜 이후로 등록이 가능하다. 

- 상품 수정 및 삭제
마이페이지에서 각 카테고리를 선택하면 회원 본인이 등록 및 판매한 상품을 조회할 수 있다. 
이 페이지에서 수정과 삭제가 가능하며, 수정은 상품 등록과 동일하게 동작한다. 
단, 공동구매는 구매자가 존재할 경우에 삭제가, 경매는 해당 상품에 입찰자가 존재할 경우에 수정과 삭제가 불가하다. 
(※주의 : 구매자나 입찰자가 존재할 경우, 삭제는 버튼을 눌렀을 때 alert 창이 뜨지만, 수정의 경우 입력 폼을 다 입력하고 등록 버튼을 눌렀을 때 alert 창이 뜬다)
또한, 판매가 완료된 상품이거나 종료된 상품은 판매 완료 및 경매 종료 상품이라고 뜨며, 수정 및 삭제가 불가하다. 
(※주의 : 상품 수정 시, 사진이 초기화되므로 사진을 꼭 다시 등록해야 한다.)

- 카테고리 검색 
상단바의 카테고리를 통해 등록된 해당 카테고리 상품들을 확인할 수 있다. 
판매 종료된 상품일 경우, 흐리게 처리하여 판매 중인 상품과 구별되어 보이고 각 상품을 클릭하면 해당 상품의 상세 화면으로 이동한다. 
직거래/공동구매는 판매 중인 상품의 경우, 해당 상품의 정보와 구매 예약 및 관심 항목을 추가할 수 있으며, 판매 종료된 상품의 경우, 판매 불가 상품이라는 안내문이 보여진다. 
그리고, 경매는 판매 중인 상품의 경우, 해당 상품의 정보와 입찰 및 관심 항목을 추가할 수 있으며, 판매 종료된 상품의 경우, 판매 불가 상품이라는 안내문과 함께 입찰 목록을 확인할 수 있다. 

- 직거래 상품 구매
구매자가 상품 상세 화면에서 구매 예약 버튼을 눌러 장바구니에 담아 놓고 거래가 완료되면 판매자가 직접 판매 완료 버튼을 눌러야만 구매 완료가 된다. 
이때, 판매 완료 버튼을 누르면 자동으로 구매 처리가 되어 구매자의 장바구니에서는 해당 상품이 사라지고 구매 내역에 저장된다.

- 공동구매 상품 구매
공동구매의 경우 구매 예약 버튼을 누르고 장바구니에 담은 뒤 장바구니에서 구매 버튼을 누르면 구매가 가능하다. 
이때, 현재 구매 인원이 최대 인원과 동일해지면 마감 시간이 되지 않아도 자동으로 닫히게 된다.

- 경매의 입찰 기능
경매의 상세 페이지에서 입찰하기 버튼을 누를 경우, confirm 창을 통해 입찰 여부를 묻는다. 
확인을 누를 경우에만 입찰이 되며, 입찰 성공 시 입찰 현황을 보여주는 표가 뜬다. 
입찰 현황을 보여주는 표 밑의 경매 홈 버튼을 누르면 경매의 전체 상품 목록 페이지로 이동한다. 

- 경매 상품 구매
경매가 종료되면 종료된 상품에 한해서 최고 입찰 금액의 대상인 회원의 장바구니에 상품이 담기게 된다. 
회원은 구매 버튼을 눌러야 최종 구매가 확정되며, 구매 내역을 통해 확인이 가능하다. 
만일, 구매를 원하지 않아 포기 버튼을 누르면 낙찰이 포기되고 입찰 현황 표에서도 사라지며, 다음 후순위자가 낙찰되어 후순위자의 장바구니에 상품이 담기게 된다. 
(※ 참고 : 장바구니에서 보이는 가격은 경매에서 낙찰 금액이다.)

- 구매 내역
마이페이지의 구매 내역을 통해 본인이 구매한 상품 내역을 카테고리별로 확인할 수 있다.
- 관심 항목 추가/삭제 및 조회
각 상품의 상세 화면에서 찜 버튼을 누르면 찜 버튼의 숫자가 올라가며, 해당 상품이 관심 항목에 추가된다. 
만약, 이미 추가된 상품일 경우, 관심 항목에서 삭제되며 버튼의 숫자가 줄어드는 것을 확인할 수 있다. 
관심 항목 조회는 마이페이지의 관심목록 확인 및 상단바 상단의 Wishlist를 통해 각 카테고리별로 확인이 가능하며, 관심 목록 삭제 버튼을 클릭하면 삭제가 가능하다.

- 감자톡
직거래, 공구, 경매 등의 상세 화면에서 판매자 프로필 우측에 감자톡 버튼을 클릭하면 판매자와 채팅을 할 수 있다.
또한, 상단바의 감자톡을 클릭하면 대화한 이력이 있는 사용자 목록이 확인 가능하다. 
채팅을 보낼 때, 아무것도 입력하지 않고 전송을 누르면 alert 창과 함께 메세지가 보내지지 않는다.

- 검색
상단바 카트 좌측에 있는 돋보기 버튼을 눌러 직거래, 공구, 경매 중 하나의 카테고리를 선택하고, 검색하고 싶은 상품을 검색하면 검색이 가능하다. 
검색 결과로는 검색어를 포함하고 있는 이름의 상품들이 나오게 된다. 만약 검색 결과가 없다면, ‘검색 결과가 없습니다’라고 안내 글이 나온다.

<< 주의 사항 >>
1. 검색 및 조회를 제외한 모든 기능은 모두 로그인을 해야 가능합니다.
* 로그인이 되어있지 않을 경우, 자동으로 로그인 화면으로 넘어갑니다.
2. 상품 등록 시 함께 등록되는 사진의 경우 프로젝트내에는 업로드가 정상적으로 되지만, 톰캣 서버가 바로 인식하지 못하여 사진 로딩이 바로 되지 않습니다. 
* 약 5초 뒤에 새로고침 버튼을 누르면 정상적으로 인식되어 사진이 로딩됩니다.
(새로고침 버튼을 눌러도 뜨지 않는다면, 프로젝트를 refresh 해주거나 
window>preferemces>general>workspace에서 Refresh using native hooks or polling 항목을 체크해주면 새로고침만으로도 사진이 로딩됩니다.)
